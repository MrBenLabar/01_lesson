My First HTML Page

June 20, 2026

Today I created my first complete HTML webpage. I learned how to structure a document using semantic HTML elements such as header, nav, main, article, section, and footer.

I also learned how to create headings, paragraphs, links, lists, tables, and images. Understanding the basic structure of an HTML document helped me see how websites are built.

One of the most interesting concepts I learned was the use of internal links, which allow users to navigate different sections of the same webpage.

Learning CSS Styling

June 22, 2026

After learning HTML, I started working with CSS to improve the appearance of my webpages. I learned how to use selectors, colors, fonts, margins, padding, and borders.

One of my favorite topics was learning how to style text using properties such as font-family, font-style, font-weight, and line-height. These properties make webpages more readable and visually appealing.

I also experimented with responsive design techniques and discovered how CSS can transform a simple webpage into a professional-looking website.
